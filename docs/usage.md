# 使用文档

## 快速开始

### 首次使用

1. **启动应用**
   ```bash
   ./scripts/dev.sh
   ```
   
2. **访问系统**
   打开浏览器访问 http://localhost:5173

3. **添加第一个账户**
   - 点击左侧「账户管理」
   - 点击「新增账户」
   - 填写账户信息并保存

## 功能指南

### 1. 账户管理

账户是财务管理的基础单元，支持以下类型：

| 类型 | 说明 |
|------|------|
| 银行卡 | 储蓄卡、信用卡 |
| 支付宝 | 余额、余额宝 |
| 微信 | 零钱、理财通 |
| 现金 | 线下现金 |
| 其他 | 其他类型资产 |

**操作步骤：**
1. 点击「新增账户」
2. 填写账户名称、类型、余额
3. 勾选是否计入净资产
4. 保存

### 2. 交易记录

记录每笔收支，支持智能分类：

**添加交易：**
1. 点击「新增交易」
2. 选择账户、类型（收入/支出）
3. 输入金额
4. 选择或输入分类
5. 添加备注、标签（可选）
6. 保存

**快捷操作：**
- 批量导入：支持 CSV 格式
- 智能分类：系统自动识别
- 标签筛选：快速定位

### 3. 基金持仓

管理基金投资，追踪收益：

**添加基金：**
1. 输入基金代码（如：000001）
2. 系统自动获取基金名称
3. 输入持有份额、成本价
4. 保存

**功能说明：**
- 实时净值更新
- 收益率计算
- 持仓占比分析

### 4. 贷款管理

追踪房贷、车贷等贷款：

**添加贷款：**
1. 填写贷款名称、类型
2. 输入本金、利率
3. 设置还款日
4. 关联还款账户

**实用功能：**
- 还款计划：查看完整还款表
- 提前还款：模拟提前还款效果
- 进度追踪：剩余本金、已还比例

### 5. 定期账单

管理周期性支出，避免遗漏：

**智能识别：**
系统自动分析历史交易，识别可能的周期账单

**手动添加：**
1. 设置账单名称
2. 选择周期（月/季/年）
3. 设置提醒天数
4. 保存

### 6. 自动化规则

设置规则自动处理交易：

**规则类型：**
| 触发条件 | 说明 |
|---------|------|
| 商户匹配 | 根据商户名匹配 |
| 金额条件 | 根据金额范围 |
| 分类匹配 | 根据交易分类 |

**执行动作：**
- 添加标签
- 设置分类
- 发送通知
- 标记待审

### 7. AI 财务助手

智能对话式财务查询：

**快捷查询：**
- 账户余额
- 本月收支
- 基金持仓
- 贷款情况
- 储蓄率
- 消费分析

**自由提问：**
直接输入问题，如：
- "我这个月在餐饮上花了多少？"
- "给我一些储蓄建议"
- "分析一下我的消费习惯"

### 8. 资产全景

一览全局资产状况：

- 总资产统计
- 资产趋势图
- 资产分布饼图
- 净资产变化

### 9. 财务健康

五维度健康评分：

| 维度 | 说明 |
|------|------|
| 储蓄率 | 收入结余比例 |
| 应急金 | 月支出覆盖能力 |
| 负债率 | 负债占资产比例 |
| 预算执行 | 预算达成情况 |
| 投资波动 | 基金收益稳定性 |

## 数据安全

### 本地存储

所有数据存储在本地 PostgreSQL 数据库，确保隐私安全。

### 数据备份

```bash
# 备份数据库
pg_dump -U postgres finance_manager > backup.sql

# 恢复数据
psql -U postgres finance_manager < backup.sql
```

## 常见问题

### Q: 如何修改账户余额？

A: 直接编辑账户，修改余额即可。系统会记录变更历史。

### Q: 交易可以删除吗？

A: 可以，但建议使用「作废」状态保留历史记录。

### Q: AI 助手数据安全吗？

A: AI 仅用于分析汇总数据，不会存储您的具体交易明细。

### Q: 如何导出数据？

A: 在「报表分析」页面，支持导出为 CSV 格式。
